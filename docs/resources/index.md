# Resources

This is a list of free resources that I've found to be really worthwhile. A lot of this is personal preference, so if there's anything you've found that you think would benefit others, please let me know about it! 

🚧 **Think of this list as a living document. This is by no means a comprehensive list of all the free resources available, but I'll do my best to update this as time allows.** 🚧
## General

<div class="grid cards" markdown>

-   ![](attachments/Pasted%20image%2020250907070854.png){ .lg .middle } 

    ---

    !!! tip inline end "If in doubt"
	    If you're overwhelmed by all the choices, start here!
    PWN.COLLEGE is an amazing free resource from Arizona State University. Topics include general computing, Linux, scripting, web security, reverse engineering, and a ton more!

    [:octicons-arrow-right-24: Go to pwn.college](https://pwn.college)

</div>

<div class="grid cards" markdown>

-  ![](attachments/Pasted%20image%2020250907082434.png){ .lg .middle width="100%" } 

    ---

    The TryHackMe free tier offers around 500 rooms to get started with IT and security. It's worth noting that there are some limitations with the free tier, but it's absolutely worth checking out if you're getting started. The premium tier is reasonably priced, and I feel like it's probably the best value for the money if you're looking to pay for content.

    [:octicons-arrow-right-24: Go to TryHackMe](https://tryhackme.com/dashboard)

</div>

<div class="grid cards" markdown>

-  ![](attachments/Pasted%20image%2020250907080042.png){ .lg .middle width="100%"}

    ---

    HackTheBox is one of the most well known platforms for cybersecurity education, and it has a free tier available. It's worth noting that HTB machines are generally considered to be harder than equivalent difficulties on other platforms, so there might be a bit of a learning curve. HackTheBox also offers an Academy, but there's an additional cost for that. The premium tier gives you access to private machine instances and other benefits, but it can get pricey once you start adding on academy modules, pro-labs, etc. Regardless, this is still a great resource to check out.

    [:octicons-arrow-right-24: Go to HackTheBox](https://app.hackthebox.com/home)

</div>

<div class="grid cards" markdown>

-  ![](attachments/Pasted%20image%2020250907072023.png){ .lg .middle } 

    ---

    OverTheWire is another free resource that is good for learning Bash, web security, and more. It's older and doesn't seem to be as up-to-date as pwn.college is, but it's still pretty decent. OverTheWire consists of a series of "War Games", each focused on a different topic. Definitely worth checking out.

    [:octicons-arrow-right-24: Go to OverTheWire](https://overthewire.org/wargames/)

</div>

    [:octicons-arrow-right-24: Go to PortSwigger Web Security Academy](https://portswigger.net/web-security)
